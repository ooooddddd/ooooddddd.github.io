# 一、安装

# 1.1 安装

  1. 如图所示，运行 docker 镜像安装 rabbitmq

  2. 报错提示：网络未找到

  3. 网络需要输入自己有的：
    docker run -e RABBITMQ_DEFAULT_USER=ZC -e RABBITMQ_DEFAULT_PASS=123456 -v mq-plugins:/plugins --name mq --hostname mq -p 15672:15672 -p 5672:5672 --network bridge -d rabbitmq:3.8-management

  4. 如何访问 http://192.168.110.92:15672/

  5. 访问成功页面

# 2.2 快速入门

  1. 添加消息队列
  2. 发送消息到交换机（exchanges），然后交换机需要 binding 消息队列，否则消息无法发送。
  3. 总结：交换机没有存储消息的能力，只能转发消息

# 2.3 数据隔离

  1. 用户管理
